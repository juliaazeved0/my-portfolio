// sobre mim
origem
Comecei como professora de robótica e educação digital na rede municipal de Foz do Iguaçu. A sala de aula foi onde descobri que queria estar do outro lado do código — construindo, não apenas ensinando.
hoje
Estagiária no Itaipu Parquetec, desenvolvendo soluções com .NET e Unity 3D para Gêmeos Digitais. Curso ADS no IFPR e contribuo como voluntária em pesquisas de ciência de dados e desenvolvimento de jogos para segurança cibernética.
Carrego da docência algo que muitos desenvolvedores levam anos para aprender: a capacidade de comunicar o complexo de forma clara, de trabalhar com empatia em equipe e de resolver problemas criativamente sob pressão. Sou movida por aprendizado contínuo, autonomia e impacto real. Minha formação em Física moldou meu raciocínio analítico — e ele acompanha cada linha de código que escrevo. Acredito que tecnologia é mais poderosa quando constrói pontes: entre dados e decisão, entre o sistema e a pessoa que ele serve.

jogo · voluntária
Cyberlace

Jogo top-down em pixel art desenvolvido como iniciativa do Ladies in Cyber no Itaipu Parquetec (dez/2025 – abr/2026), com o objetivo de aumentar a participação feminina em CTFs de segurança cibernética. Gamifica desafios reais de OSINT, Linux, engenharia social, forense digital e criptografia.

Desenvolvi a aplicação completa com autonomia total, em ciclos quinzenais de entrega, usando metodologias ágeis com Trello, Notion e Miro.

pesquisa · voluntária
Mapeamento de Stack — Gêmeos Digitais Web

Pesquisa para apoiar a tomada de decisão tecnológica no Itaipu Parquetec: mapeei a stack de projetos de Gêmeos Digitais na Web com Unity como engine de visualização. Usei Python e Google Colab para coletar repositórios via API do GitHub com Google Apps Script, e o ASReview com machine learning para triagem e seleção de artigos científicos relevantes — estruturando indicadores sobre tecnologias, frameworks e protocolos adotados na área.

Python ASReview Google Colab Jupyter Apps Script Google Sheets GitHub API Machine Learning
